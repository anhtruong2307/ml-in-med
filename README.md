# Heart Disease Classification

This project uses machine learning models to predict heart disease from a dataset stored in `heart.xlsx`.  
The models used are Naive Bayes, SVM, and Random Forest, with PCA applied for dimensionality reduction.

## Files
- `heart.xlsx` - dataset
- `pca_nb_svm_rf.py` - Python script
- `PCA+NB,SVM,RF.ipynb` - Jupyter Notebook
- `final_ml_in_med_Group1.pdf` - final report
